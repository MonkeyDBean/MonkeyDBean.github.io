参数解读：
-d ： 后台运行

–name ： 命名容器名

-p 6379:6379 ： 指定端口映射，前一个6379为宿主机端口号，后一个6379为容器端口号

-v $PWD/data:/data ： 将主机中当前目录下的data挂载到容器的/data

–requirepass root : 设置密码为root，保护 Redis 服务器

–redis-server –appendonly yes :在容器执行redis-server启动命令，并打开redis持久化配置
